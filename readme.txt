Add Simple History integration for NextCellent.

== Description ==

NextCellent Simple History logs events related to NextCellent in the [Simple History](https://wordpress.org/plugins/simple-history/) plugin.

Currently, it logs following events:

* Adding galleries
* Updating galleries
* Deleting galleries
* Adding pages for galleries
* Adding images to galleries
* Adding albums
* Editing albums
* Deleting albums
* Updating the NextCellent options
* Deleting images
* Updating images

Note: this only works with NextCellent Gallery, not with NextGEN Gallery.

#### Languages

So far NextCellent Simple History is available to:

* Dutch
* English
* Polish (thanks to [Frank P. Walentynowicz](http://fw2s.com/))
* Danish ([thomasdk81](https://profiles.wordpress.org/ThomasDK81))

If you want to help, please contact me.
